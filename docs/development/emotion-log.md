- こんな初歩的なミスでAPIサーバーに負荷かけるとか最悪

### 状況

- プロンプト生成がAPIを叩き続ける無限ループ発生

- 依存配列を空配列`[]`に変更
- 初回のみ実行されるように修正
- 状態チェックも追加して安全性向上

### 反省

- React Hooksの基本を忘れてた
- コードレビューで見逃されてたのも問題
- 自動生成機能を有効化する時にちゃんとテストすべきだった
